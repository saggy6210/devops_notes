Linux & Bash Interview Questions

1

How to check the default route and routing table?

netstat -rn
route -n
ip route show
2

How to check which ports are listening on my Linux server?

netstat -tulpn
ss -tulpn
lsof -i
3

How to print a specific line and convert it to uppercase?

awk 'NR==line-number{print toupper($0)}' filename

Example: Print line 5 in uppercase:

awk 'NR==5{print toupper($0)}' myfile.txt
4

How to display the number of users logged into the server?

w
who
users | wc -w
5

How to print date in different formats?

date +%Y%m%d%H%M%S # 20240115143022
date +"%Y-%m-%d" # 2024-01-15
date +"%d/%m/%Y" # 15/01/2024
6

How to check if a specific process is running?

If return value is >0, process is UP:

ps -fu username | grep "process_name" | grep -v grep | wc -l

Or using pgrep:

pgrep -x process_name
7

How to delete files older than a specific number of days?

find . -type f -name "*.log" -mtime +20 -exec rm -f {} \;

This deletes .log files older than 20 days.

8

How to check server uptime?

uptime

Output shows current time, how long system has been running, number of users, and load averages.

9

How to check RAM size of the server?

free -h # Human readable
free -g # In GB
cat /proc/meminfo
10

How to replace content/string in a file?

Using sed:

sed -i "s/old_pattern/new_pattern/g" file

Using perl:

perl -pi -e 's/old_pattern/new_pattern/g' file
11

How to setup passwordless SSH connectivity?

  1. Generate SSH key pair: ssh-keygen -t rsa
  2. Copy public key to remote: ssh-copy-id user@remote_host
  3. Or manually copy ~/.ssh/id_rsa.pub to remote's ~/.ssh/authorized_keys
12

How to copy files from a remote server?

Single file:

scp user@remote:/path/file local_path

Directory (recursive):

scp -r user@remote:/path/dir/ local_path
13

What is the difference between soft link and hard link?

  • Hard link: Points directly to inode, same file. Deleting original doesn't affect hard link.
  • Soft link (symlink): Points to filename. Becomes broken if original is deleted.
ln file hardlink # Hard link
ln -s file softlink # Soft link
14

How to find files by name or pattern?

find /path -name "filename"
find /path -name "*.log"
find /path -type f -name "*.txt"
15

How to check disk usage?

df -h # Filesystem usage
du -sh /path # Directory size
du -sh * # Size of all items
16

What are file permissions in Linux?

Three types: r (read=4), w (write=2), x (execute=1)

Three levels: Owner, Group, Others

chmod 755 file # rwxr-xr-x
chmod 644 file # rw-r--r--
chown user:group file
