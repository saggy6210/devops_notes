Introduction & Installation

🚀 What is Ansible?

Ansible is an open-source automation tool for configuration management, application deployment, and orchestration. It uses simple YAML files (playbooks) to describe automation tasks and is agentless, connecting over SSH.

Installation

  • Linux: sudo apt install ansible or sudo yum install ansible
  • macOS: brew install ansible
  • Python/pip: pip install ansible
Check Version 
ansible --version

Inventory Basics

Ansible uses an inventory file to define hosts and groups:

Sample Inventory 
[web]
web1.example.com
web2.example.com

group1:children
web
db

Default location: /etc/ansible/hosts (can use -i flag for custom files)

First Ad-hoc Commands

Run simple commands on remote hosts without a playbook:

Ping All Hosts 
ansible all -m ping
Run Shell Command 
ansible web -m shell -a 'uptime'
