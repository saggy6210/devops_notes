Day 07: CloudFormation & CDK

CloudFormation

  • Infrastructure as Code (IaC) for AWS resources.
  • YAML/JSON templates to define and provision infrastructure.

CDK (Cloud Development Kit)

  • Define cloud infrastructure using familiar programming languages.
  • Supports TypeScript, Python, Java, etc.

Hands-on

  1. Write a simple CloudFormation template.
  2. Deploy a stack using CDK.

Interview Tips

  • Compare CloudFormation and CDK.
  • Discuss benefits of IaC.
