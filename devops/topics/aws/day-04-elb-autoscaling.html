Day 04: ELB & Auto Scaling

ELB (Elastic Load Balancer)

  • Distributes incoming traffic across multiple targets (EC2, containers).
  • Types: Application, Network, Gateway Load Balancers.

Auto Scaling

  • Automatically adjusts number of EC2 instances based on demand.
  • Scaling policies, launch configurations, target groups.

Hands-on

  1. Set up an Application Load Balancer.
  2. Configure an Auto Scaling group.

Interview Tips

  • Explain difference between ELB types.
  • Discuss scaling strategies and cost optimization.
