Day 09: Security Best Practices

Security in AWS

  • Shared responsibility model: AWS secures the cloud, you secure what you put in it.
  • IAM, encryption, network security, monitoring.

Best Practices

  • Enable MFA, use strong passwords, rotate keys.
  • Least privilege principle, audit logs, patch management.

Hands-on

  1. Enable encryption for S3 and RDS.
  2. Review IAM policies and CloudTrail logs.

Interview Tips

  • Discuss security tools and compliance in AWS.
  • Explain how to secure access to resources.
