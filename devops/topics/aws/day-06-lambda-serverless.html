Day 06: Lambda & Serverless

AWS Lambda

  • Run code without provisioning servers (serverless).
  • Event-driven, supports multiple languages.
  • Triggers: S3, API Gateway, CloudWatch, etc.

Serverless Concepts

  • Benefits: scalability, cost, no server management.
  • Other services: API Gateway, Step Functions, DynamoDB.
